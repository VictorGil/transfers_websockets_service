# Transfers Websockets Service
This is an example/PoC of how to extend the event-driven paradigm all the way to the end user's
Web browser using WebSockets.  
  
## Requirements
 - Maven 3  
 - Java 12  
 - Confluent Platform (which includes Kafka)
